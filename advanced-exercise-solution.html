Vad är semantisk HTML?

Semantisk HTML handlar om att använda HTML-element som beskriver innehållets betydelse och struktur för både webbläsare och människor.

Fördelarna med semantisk HTML

Genom att använda semantiska element som <article>, <nav> och <footer>, kan du göra din webbplats mer tillgänglig för användare med funktionsnedsättningar och förbättra dess SEO.
